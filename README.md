# social-media-sql
# Social Media Trends Analysis (SQL + Power BI)

## Project Overview
This project analyzes 5,000 social media posts using PostgreSQL and Power BI.

## Tools Used
- PostgreSQL
- SQL
- Excel
- Power BI

## Data Cleaning
- Imported CSV
- Checked duplicates
- Checked missing values
- Converted date format

## SQL Analysis
- Total Views
- Total Likes
- Views by Platform
- Top 10 Hashtags
- Region Analysis
- Monthly Trends
- Window Functions (ROW_NUMBER)

## Dashboard
- KPI Cards
- Platform Analysis
- Region Analysis
- Content Type Analysis
- Monthly Trend
- Top Hashtags

## Key Insights
- TikTok generated the highest views.
- #Challenge was the most viewed hashtag.
- India and the UK had the highest engagement.
